**이평 수렴 계산:**
```python
def check_convergence(closes, threshold=3.0):
    ma5 = sum(closes[:5]) / 5
    ma20 = sum(closes[:20]) / 20
    ma60 = sum(closes[:60]) / 60
    spread = (max(ma5,ma20,ma60) - min(ma5,ma20,ma60)) / closes[0] * 100
    return spread < threshold
```
