# 수출 데이터 활용 프레임워크 (테린이 아빠 영상 정리)

> 2026-04-23 정리. 향후 **관세청/산자부 수출 데이터 봇 통합** 시 설계 참조용.
> 출처: 유튜브 "테린이 아빠" 채널 (2~3년 시행착오 경험담)

---

## 1. 데이터 출처 & 3가지 조회 방식

**접속**: 구글/네이버 "트레이드스(TRASS)" 검색 → 무역통계 → 수출입통계

| 조회 유형 | 내용 | 발표 시점 | 특징 |
|---------|------|---------|------|
| **잠정치 조회** | 전국 단위 총액 | 매월 **1일, 11일, 21일** | 10단위 기본, **유료(~33,000원/월)** 시 4·6단위 |
| **기본 조회** | 확정 수출 데이터 | 매월 **15일** (공휴일도 10시경) | 전월 최종치 |
| **상세 조회** | 품목 × 지역 × 국가 교차 | — | **유료 전용**, 기업 추적에 핵심 |

### HS 코드 단위
- **10단위**: 세부 품목 (잠정치 기본)
- **6단위**: 상위 묶음 (시장 실무 표준)
- **4단위**: 대분류
- 시장은 **6단위·10단위** 관리, 최소 4단위

---

## 2. 데이터 관리 방법

### 월 → 분기 변환
- 엑셀에 **영업일수** 미리 기재
- **일평균(1평균) 수출액** 산출
- 일평균 3개월 합 = **분기 평균**
- 비교축: **MoM / YoY / QoQ**

### 판가(ASP) 역산
> **수출금액(달러) ÷ 중량 = 판가**
> 
> 애널리스트 컨센서스 판가와 대조 가능

### 핵심 판단 기준
> **"이번 달이 직전 3개월 평균보다 높은가?"**
>
> 한 달 튀는 숫자에 흔들리지 말고 **분기 트렌드**로 본다. 
> 전월보다 낮아도 직전 분기 평균 위라면 QoQ 개선 방향.

## 3. 잠정 vs 확정

### 잠정치가 우선
- 과거엔 확정치(15일) 발표마다 시장 흥분 → **지금은 무뎌짐**
- 이미 잠정으로 다 반영된 뒤의 확정치는 **후행 정보**

### 확정치 필요 케이스 = 지역별 분해 필요 품목
잠정은 전국 단위라 특정 기업 실적 발라내기 어려움.

| 유형 | 사례 | 필요 분해 |
|------|------|----------|
| **단일 기업 지배** | 라면 → 삼양식품 | 잠정만으로 충분 |
| **지역 집중** | 미용기기 901890 → 원텍(대전 유성), 클래시스 | **지역별 쪼개기 필수** |
| **대표 생산지** | 라면 → 원주·밀양 합산 | 지역 합산 |

**원칙**: 잠정에서 안 보이는 것만 확정치로 보완

---

## 4. 업종별 해석 반전

수출 증가 = 무조건 호재가 **아님**.

### 반도체형 (반대로 봐야) ⚠️
- YoY **고점 → 피크아웃 → 주가 하락**
- 최악 진입 → YoY 개선 시작 → **주가 반등**
- "수출 좋음 = 매수" 아닌, **변곡점** 확인

### 일반 소비재형
- YoY 개선 지속 = **순방향 호재**

### 애널 보고서 교차 확인 필수
> "2분기부터 수출 확대될 것"이라 써있는데 1~3월 잠정치 부진에 실망해서 팔면 **선반영 구간 놓침**.

---

## 5. ⭐ 핵심 결론 — 수출 데이터는 "메인"이 아니다

### 저자 깨달음 전환
> 처음: "이거 잘 정리하면 노다지네!"
> 지금: "수출 데이터는 **모멘텀 중 하나**일 뿐"

### 수출 데이터 = 재료 1/5
- 주가 모멘텀
- 동종 기업 주가 흐름
- 컨센서스 상향
- **수출 데이터**
- 수급 상태

### 선택과 집중
> **"시장의 중심주만 봐라"**

**중심주 판별 교집합**:
1. **상대강도(RS)가 센 섹터**
2. **유동성이 계속 공급되는 섹터** (시총/거래대금 등수)
3. **수급이 비어 있는 종목**

이 교집합 밖 변두리는 아무리 수출 좋아도 **확률 낮음**.

---

## 6. 수출 좋아도 바로 들어가면 안 되는 이유

### 현대일렉트릭 사례
- 고용량 변압기 수출 데이터 👍
- 그러나 주가는 **과열권**
- → 수출 데이터는 "확인"만. **수급 비어질 때까지 한 턴 기다림**

### APR (뷰티 디바이스) 반대 사례
- 저자의 뷰티 디바이스 수출 데이터 = 부정적
- 애널은 **화장품 호조** 전망
- APR의 화장품만 수출로 발라내기 어려움
- 주가는 상승
- → **"내 한 가지 시각이 틀렸을 수 있다"** 인정

## 7. 실전 체크리스트 (의사결정)

| # | 확인 항목 |
|---|---------|
| ① | 시장 중심주 섹터인가? (RS + 유동성 교집합) |
| ② | 잠정 수출 데이터 QoQ·YoY 방향성 |
| ③ | 업종 특성 (순방향 vs 변곡점형) |
| ④ | 애널리스트 컨센서스/보고서 일치 여부 |
| ⑤ | **수급이 비어 있는가** (과열 여부) |
| ⑥ | 확정치 필요 품목이면 지역별 분해 |

**배우자 고를 때 한 가지만 보지 않듯** 주식도 수출 데이터 하나만 보지 않음.

## 봇 통합 설계 반영 사항 (향후 구현)

### Phase 1: TRASS 잠정치 자동 수집 (무료)
- 출처: TRASS 잠정치 페이지 (매월 1/11/21일)
- 범위: 전국 단위, HS 10단위, 품목 전체
- 계산: 월 → 일평균 (영업일수 보정) → 분기합산 → YoY/MoM/QoQ
- 공수: 6~8시간 (TRASS HTML/엑셀 파싱)

### Phase 2: 알파 통합 (핵심)
**스캐너 프리셋 `export_momentum`**:
```
수출 QoQ 개선
AND 상대강도(RS) 상위
AND 거래대금/시총 유동성 유입
AND 최근 수급 비어있음 (foreign/inst 순매수 관망)
→ 교집합만 추려냄
```

**MCP 도구 `get_export_trend(hs_code, category, periods)`**:
- YoY/MoM/QoQ 3축
- 판가(ASP) 역산 + 컨센 대조
- 업종별 해석 태그 (반도체=역해석 / 소비재=순방향)

### Phase 3 (선택): 유료 구독
- TRASS 유료 → 지역 × 국가별 분해
- 원텍·클래시스 / 삼양식품 등 기업 추적
- 월 3.3만원 투자 가치는 2~3개월 써보고 결정

---

## 기존 투자룰과 연결

- **"RS + 유동성 + 수급 비어있음" 교집합** = A등급 조건 강화에 활용
- **"수출 좋아도 과열이면 한 턴 기다려라"** = 3-tranche 분할매수 + RR 알림 정합
- **"피크아웃형 업종은 YoY 반대"** = 반도체(삼성/하이닉스) 재진입 필수 체크
- **"시장 변두리는 버려라"** = 테마주 회피 원칙 동일

## 어제 영상 vs 오늘 영상 비교

- **어제**: 수출 데이터 **어디서 보고 언제 보는가**
- **오늘**: 수출 데이터를 **어떻게 해석하고 무엇과 교차**시킬 것인가

→ 오늘 프레임워크가 **한 단계 깊음**.
