# REGIME_DESIGN.md — 시장 레짐 재설계 (한국형/미국형 분리)

> **확정 근거**: `REGIME_RESEARCH_BRIEF.md` 딥리서치 회신 (2026-06-07). H1~H6 실증 판정 기반.
> **절대 제약**: 레짐의 유일한 용도 = **통화별 현금 다이얼**(🟢/🟡/🔴). 종목 매수/매도/손절 게이팅 금지(INVESTMENT_RULES §5, 매크로=현금·동결 도구).
> **현금 철학**: 🟢 평상 5~8%(상비) / 🟡 경계 8~15% / 🔴 위기 **풀투자 지향(dry-powder 발사)**. 🔴는 "현금 쌓기"가 아니라 "실탄 발사 국면".
> **상태**: 설계 확정. 백테스트(아래 §5)로 한국형 검증 후 구현(Claude Code).

## 0. 핵심 결론 (왜 한국형을 따로 만드나)

**미국 방식(S&P 200MA + VIX)을 한국에 복사 금지 — 실증 지지.**
- KCMI 보고서 856호(2017): KOSPI 2011 하반기~2017.5 "BOXPI" 박스권(약 1,800~2,100, 6년 횡보) → 200MA crossover는 whipsaw 양산.
- Hsu·Hsu·Kuan(2010, *J. Empirical Finance* 17(3):471–484): 16,000+ 기술룰을 미국 3개·신흥국 6개 지수(KOSPI 포함)에 검증 → 단순 추세룰이 buy-and-hold 미달.
- → **한국형 = 변동성(VKOSPI 퍼센타일) 주신호 + 추세(200MA 거리, 완충대) 보조.**

미국형은 학술 검증됨(Faber 2007: DD 46%→<10%) → **유지하되 VIX 임계만 퍼센타일화.**

## 1. H1~H6 실증 판정표

| 가설 | 판정 | 핵심 근거 | 출처 |
|------|------|----------|------|
| H1 KOSPI 200MA crossover | **기각** → 거리기반 대체 | BOXPI 6년 횡보; 16,000룰 buy&hold 미달 | KCMI 856(2017); Hsu·Hsu·Kuan(2010) |
| H2 VKOSPI vs 실현변동성 | **VKOSPI 지지** (adj R²≈0.40) | 내재변동성 예측력 최고. 단 2009~ | KOSPI200 IV 연구; 유진투자(2026, 신뢰도 중) |
| H3 외국인 순매도 선행성 | **조건부 기각** (반응이지 원인 아님) | 매도 후 음의 초과수익 없음; 2026 외인 103조 순매도에도 사상최고 | Choe·Kho·Stulz(1999, JFE 54:227); Yang(2017) |
| H4 USD/KRW | **동행 지지, 선행 약함** | KRW는 VIX 민감, 위기 동행. AI붐기 상관 약화 | AMRO(2022) |
| H5 고정 vs 퍼센타일 임계 | **퍼센타일 지지** | 분포 이동, spike 빈도 증가, 절대치 출처충돌 무력화 | RegimeFolio(2025) |
| H6 미국형(S&P+VIX) | **유지** + 임계 퍼센타일화 | DD 46%→<10%; 단순 룰 > Markov(과적합) | Faber(2007, JWM 9(4)) |

## 2. 한국형 레짐 스펙 (원화 sleeve 현금 다이얼)

**지표 (주 1 + 보조 3):**

| 구분 | 지표 | 데이터소스 | 폴백 |
|------|------|-----------|------|
| **주: 변동성** | VKOSPI | KRX/투자전문기관 | 실현변동성 (20일·60일 std, FDR `^KS11`) |
| 보조: 추세 | KOSPI vs 200MA 거리% (완충대 ±5%) | FDR `^KS11` | 없음 |
| 보조: 수급 | 외국인 5일 누적 순매도 | KRX / 봇 `get_supply` | 🔴 **확인용만** (단독 트리거 금지) |
| 보조: 환율 | USD/KRW 60일 변화 | FDR | 🔴 **확인용만** |

**퍼센타일 임계 (트레일링 252거래일):**

| 레짐 | VKOSPI %ile | KOSPI 200MA 거리 | 현금 태세 |
|------|-------------|------------------|-----------|
| 🟢 평상 | < 50%ile | > -5% (완충대 위) | 원화 5~8% |
| 🟡 경계 | 50~80%ile | -5% 통과 | 원화 8~15% (실탄 비축) |
| 🔴 위기(발사) | > 80%ile (또는 절대 VKOSPI 40+) | < -10% 동반 | 원화 풀투자 지향 (보유 A급 추가) |

**판정 로직 (변동성 우선):**
1. VKOSPI > 80%ile → 🔴 (발사). 외국인 순매도 OR USD/KRW 급변 동행 시 신뢰도↑ (확인용).
2. VKOSPI 50~80%ile AND 200MA 거리 하향(-5% 통과) → 🟡.
3. 둘 다 평온 → 🟢.
- 외국인/환율은 **점수에 안 넣음**. 🔴 확정의 확인 게이트로만 (H3·H4 결론).

**디바운스 (현 버그수정 유지 — 같은 날 중복호출 카운트 누적 금지):**
- 🔴 진입: **2거래일 연속** 확인 (빠르게 — 발사 기회 놓치지 않게)
- 🟢 복귀: **5거래일** (느리게 — whipsaw 방지)
- 🟡: 즉시

## 3. 미국형 레짐 스펙 (달러 sleeve 현금 다이얼)

**현행 유지 + 2개 개선:**
- (a) VIX 20/30 **고정 임계 → 트레일링 252일 퍼센타일** (예: 67/90%ile)
- (b) S&P 200MA 거리 ±3% 완충대 — 유지 (적절)
- VIX3M term-structure backwardation 조건 — 유지 (우수)
- 판정/디바운스 구조 동일, 현금 태세는 달러 sleeve에 매핑

## 4. 충돌 3곳 정정 (구현 시 함께) — 시장별 분리 + 새 철학

현 봇이 레짐으로 **종목 판단을 게이팅**하는 3곳을 시장별 분리 + 🔴=발사로 정정:

| 위치 | 현재 (구세대) | 정정 |
|------|--------------|------|
| `jobs/regime.py` 전환 플레이북 | "🔴: 현금 25%↑ / 손절 15→10 타이트 / B급 트림" | **현금 다이얼만** (🔴=발사·풀투자 지향). 종목 지시(손절/트림/감시가) 삭제. KR/US 별도 메시지 |
| `kr_summary.py:339` 현금권고 | "crisis & cash<25 → 25% 권장" | KR 레짐 기준 새 다이얼 (🔴=풀투자 지향) |

## 5. 백테스트 계획 (구현 전 검증 — 봇 `get_backtest` + 외부)

**목적**: 리서치 설계가 "느낌"이 아니라 우리 데이터로도 유효한지 확인. 특히 H1(200MA whipsaw)을 KOSPI에 직접 재현.

**데이터**: FDR `^KS11`·USD/KRW(1995~), VKOSPI(2009~ 가능 시), `^GSPC`·`^VIX`. 봇 `get_backtest` Y2/Y3 + 외부 장기.

**낙폭 기준점**: 2008 금융위기(KOSPI 연 -40.7%), 2011 유럽위기, 2015-16 차이나쇼크, 2018 Q4(-17%대), 2020 코로나(3월), 2022 긴축, 2024.8.5 엔캐리(-8.8% 단일일·서킷브레이커), 2025.4 관세(VKOSPI 44.23).

**검증 항목:**
1. **H1 직접 재현**: KOSPI 200MA(ma_cross) whipsaw 횟수 vs buy-and-hold (200MA crossover가 정말 부적합한가)
2. VKOSPI 80%ile 돌파가 KOSPI 고점 대비 선행/동행 일수
3. 200MA 거리 -10% 돌파 타이밍
4. 외국인·환율 동행 여부

**성공 기준:**
- (a) 🔴 적중률 = 실제 -15%+ 하락을 🔴로 잡은 비율
- (b) whipsaw 횟수 = 5일 내 🔴↔🟡 반전 (최소화)
- (c) MDD 감소 = 🟡 비축→🔴 발사 시뮬 회복수익 vs buy-and-hold
- (d) 오경보율 = 🔴 후 하락 없던 비율

**⚠️ 봇 get_backtest 한계**: 종목 전략 백테스터(ma_cross 등)지 "레짐→현금다이얼" 백테스터 아님. → H1 약식 재현은 지금 가능. 레짐 다이얼 풀 백테스트는 Claude Code 구현 필요.

## 6. 최종 권고 (리서치 + 우리 판단)

**단순 룰 유지.** 근거: ① 현금 다이얼 단일 용도엔 복합점수 과적합, ② Korea-specific 학술증거가 복잡 추세룰 미지지, ③ 변동성 1개(VKOSPI %ile) + 추세 1개(거리·완충대) + 디바운스 2지표면 충분. 외국인/환율은 확인용 게이트로만. KR/US 각 통화 sleeve 독립.

## 7. Caveats (구현·검증 시 주의)

- KOSPI에 Faber식 200MA의 MDD 감소 수치를 직접 보고한 학술 논문 **없음(확인 불가)** → 미국 수치 전용 금지, 자체 백테스트로 생성.
- VKOSPI 2009.4~ → **2008 백테스트는 실현변동성 폴백 필수**.
- VKOSPI 절대 최고치 출처 충돌(KRX 89.3 vs 한경 71) → **퍼센타일로 무력화** (H5 타당성 보강).
- 외국인 수급 의미 2024~26 구조변화(국내자금 흡수)로 약화 → 보조 가중치 낮게.
- 증권사 리서치(유진투자 등) = 비peer-review(신뢰도 중). KAIST vs KBM Journal 한국 TSM 결론 상충 → 듀얼모멘텀 취약 취급.
